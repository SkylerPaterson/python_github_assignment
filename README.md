# Study Time Tracker

This program asks the user how many hours they studied today and calculates their estimated weekly study time. It helps students understand their habits and stay aware of their productivity over a full week.

## How to Run
Open a terminal in the project folder and run: python app.py

## Example Output
Welcome to my Python program!
How many hours did you study today? 2
You are on track to study 14.0 hours this week.
